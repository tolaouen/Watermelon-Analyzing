# Watermolon System Analysis
Flask Microframework Build Web application system analyzing

## Features

- Forms Validation 
- Authentication (`Flask-Login`)
- Models Table Relationships and storing data
- Templetes UI (`HTML` `CSS` `Javacript`)
- Services Handle (CRUD)
- Routes Handle Connect Between Frontend and Backend
- Database connection (`Sqlite`)

## Setup

1. `python3.14 -m vevn .venv` or `python -m venv .venv` create virtual environment
2. Activate virtual environment:

    - `source .venv/bin/activate` for Mac & Linux
    - `source .venv/scripts/activate` for Window

3. `pip install -r requirements.txt` Insall all package 
4. `.env` encrypt file
5. Update `.env` with your DB URL and SECRETE KEY
6. `.gitignore` tools developer
